■はじめてのHTML

#3 タグでマークアップしてみよう
    <h1></h1>   ：見出し(heading)タグ
    <p></p>     ：段落(paragraph)タグ

#4 imgタグで画像を表示しよう
    <img>       ：画像(image)タグ   ※囲むものがないため閉じタグが無い
        (例)
        <img src="C:/Users/RYOSUKE/Desktop/freelance/Webサイト制作/img/face.jpg" width="120" height="120" alt="アイコン">
            属性：タグに付けるオプション情報のこと
                src     ：画像の参照元
                width   ：画像の幅
                height  ：画像の長さ
                alt     ：画像が上手く表示されなかった時の代替テキスト

#6 文書全体につけるタグを見ていこう
    <!DOCTYPE html>     ：html文書であることを示すタグ（お作法）
    <html></html>       ：ここからここまでがhtml文書ですよを示すタグ
        lang    ：言語(language)の指定    ex. 日本語　ja 
    
    <head></head>       ：この文書自体に関する情報
    <body></body>       ：文書の本文

#7 
    <meta>          ：その文書に関する情報（メタ情報）を指定して、ブラウザや検索ロボットに知らせるためのタグ
        charset     ：文字コード(コンピュータが使う文字セット)の設定、近年はほぼutf-8に統一されつつある
        name        ：文書の説明やキーワードを指定　※ここで指定した値の多くは検索エンジンで使用されるので慎重に言葉を選ぶこと
            description ：その文書の説明
        content     内容

    <title></title> ：文書のタイトル、ブラウザのタブに反映される
    <link>          ：その文書ファイルと関連する別の文書ファイルを指定し、 その関係を定義するもの
        rel         ：この文書からみた別の文章との関係
        href        ：Hypertext Reference、関連文書のURL

#9 コメントタグを使ってみよう
    <!---->     ：コメントタグ、説明書きなどに使う、サイト上には表示されない、「Ctrl + /」でコメントのON/OFF切り替えが可能
        (例)
        <!--ここから活動紹介-->

#10 
    <ul></ul>      ：unordered list、順番の無いリスト、箇条書き
    <li></li>      ：list item

#13 外部サイトへのリンクへのリンクを設定しよう
    <a></a>         ：アンカータグ、Webページにリンクを張り、そのページとリンク先のページを繋ぐ
        target      ：リンク先の文書を表示させるフレーム名
                        ★target=”_blank”だけでは危険、rel=”noopener”を一緒につけるなど対策を。
                        https://www.1st-net.jp/blog/2019/03/03/%E3%80%8Ctarget_blank%E3%80%8D%E3%81%AE%E5%8D%B1%E9%99%BA%E6%80%A7/
